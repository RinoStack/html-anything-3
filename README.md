# HTML Anything

The agentic HTML editor — your local AI agent writes the HTML, you ship it.

## Features

- 75+ AI skills
- 9 output surfaces
- Magazine, deck, poster templates
- Social media integration
- Zero API key required

## Surfaces

- Magazine
- Deck
- Poster
- XHS/Tweet
- Prototype
- Data report
- Hyperframes

## Getting Started

```bash
npm install
npm run dev
```
